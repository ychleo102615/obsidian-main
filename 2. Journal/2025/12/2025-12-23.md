## 每日精華

```ts
const ARR = ['a', 'b', 'c'] as const 
const OBJ = { A: 'a', B: 'b' } as const 
```

| 想要什麼     | 寫法                               | 結果                  |
| -------- | -------------------------------- | ------------------- |
| 陣列元素     | `(typeof ARR)[number]`           | `'a' \| 'b' \| 'c'` |
| 物件 key   | `keyof typeof OBJ`               | `'A' \| 'B'`        |
| 物件 value | `(typeof OBJ)[keyof typeof OBJ]` | `'a' \| 'b'`        |
| 特定欄位     | `(typeof OBJ)['A']`              | `'a'`               |
