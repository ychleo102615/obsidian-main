## 每日精華

[JS Objects: De"construct"ion](https://davidwalsh.name/javascript-objects-deconstruction)
文章在說 js 的 constructor 檯面下做太多太複雜的事情了。它提供了一種更簡單的模型。這與我們之前 lua class 的實作是類似的。但是用起來就會比較冗長。
```js
var Foo = {
    init: function(who) {
        this.me = who;
    },
    identify: function() {
        return "I am " + this.me;
    }
};

var Bar = Object.create(Foo);

Bar.speak = function() {
    alert("Hello, " + this.identify() + ".");
};

var b1 = Object.create(Bar);
b1.init("b1");
```
