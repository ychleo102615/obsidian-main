我認爲這題很重要。很單純，技巧很優雅。我挺納悶 neetcode 上竟然沒有。

這題是在解週賽 [[3811. Number of Alternating XOR Partitions]] 時，Claude 建議練熟的基本題。

觀念：
```
	prefix :       ---------------------
	prev Prefix:   ------------
	target range:              ---------
```

prefix 代表的可以 sum 或是 XOR 結果。透過 `prefix - target = prevPrefix` ，找出：如果 prevPrefix 曾經出現的話，就可以利用該資料。

在這題中，存的是 subarray 的數量，「subarray sum 為 **特定數字**時」的數量
