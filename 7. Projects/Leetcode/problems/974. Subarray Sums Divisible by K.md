# [974. Subarray Sums Divisible by K](https://leetcode.com/problems/subarray-sums-divisible-by-k/) #card
^1768742954648

這題關於取餘數的觀念很重要。

首先考慮正負數的情況。
再來是 prefix 算式我背搞糊塗了。

```
一般的概念：
prefix = need + target

此題：
sum[0:i] % k = need + target
target 可以是 k * n, n 是整數

轉換 need 時，我搞錯成 (k - sum) % k
但應該是：

need = prefix - target
     = sum[0:i] % k - target % k
	 = sum[0:i] % k
	 
因為 target % k = 0
```
