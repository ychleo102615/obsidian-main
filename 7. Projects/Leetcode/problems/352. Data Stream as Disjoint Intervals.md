# [352. Data Stream as Disjoint Intervals](https://leetcode.com/problems/data-stream-as-disjoint-intervals/) #card
Binary Search

這題是在解 paypay card code signal 第四題之後，去找類似題型找到的。是 hard 的題目。
我認為該題困難的地方在於實作 binary search。

我覺得很可惜。我知道 binary search 要怎麼實作，但是時間太趕了。我理解題目的速度太慢了。
因為我基本上昨天才剛了解 binary search 要如何應用到各種場景、否定式確認等。


把要問的問題，從找「與 value 最靠近的 interval」，變成找「大於等於 value 的 interval」。

這個大於等於怎麼定義呢？
等於：value 在 interval 裡面
大於：value > interval.start
