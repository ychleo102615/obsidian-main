nums = [str(num) for num in nums]
def cmp(a, b) -> int:
	an = len(a)
	bn = len(b)
	ai = bi = 0
	while True:
		if ai == an and bi == bn:
			return 0
		if ai == an:
			ai = 0
		if bi == bn:
			bi = 0
		if a[ai] != b[bi]:
			return ord(a[ai]) - ord(b[bi])

		bi += 1

這題一開始還想著要從數字計算下手，後來想到先轉成字串，從頭開始比對比較簡單。

一開始的想法遇到了細節處理的錯誤：
```
a = 53
b = 535
當 a.length == i時，這個比較等意於
a = 53, b = 5
因爲結果必為53開頭
```
沒處理到長度為零的情況。
```
[8308,8308,830]
```

之後看了解答才莞爾，比較直接用`a + b > b + a ? -1 : 1`就解決了。但是執行速度較慢的樣子。
