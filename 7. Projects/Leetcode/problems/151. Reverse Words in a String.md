這題應該算簡單的題目，用typescript寫甚至只要一行。
```ts
function reverseWords(s: string): string {
	return s.trim().split(/\s+/).reverse().join(" ");
};
```

練習到split中要用regexp的話，要用兩個除號包起來。然後trim可以消除頭尾空白（查chatgpt）。
