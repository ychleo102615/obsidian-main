> [!NOTE] Binary Heap
> Binary Heap (二元堆積) 是一種常見的資料結構，適合需要取最大最小值的場合，也適合用來解決 top-k 問題，同時也常被用來實作 priority queue (優先權佇列)

這題可以作為實踐binary heap的練習。
