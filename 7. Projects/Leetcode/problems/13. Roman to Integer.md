花比較多的時間在搜尋ts map的宣告用法，和string to array的用法。

# map declaration
```ts
const map = new Map<string, number>([
	["A", 123],
	["B", 456]
]);
```

跟lua的差異，可以發現類別宣告中，整個是用方括號，裡面用tuple的方式去描述一組key value的組合。


# string to array
```ts
const array = str.split('')
```
