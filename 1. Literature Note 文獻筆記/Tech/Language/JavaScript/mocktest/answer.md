##### Q5
`Object.defineProperty`
三個參數：
- **obj**：目標物件。
- **prop**：要定義或修改的屬性名稱（字串或 Symbol）。
- **descriptor**：屬性描述符，用來設定該屬性的「特權」。

| 特徵           | 說明                               | 預設值       |
| ------------ | -------------------------------- | --------- |
| value        | 實際數值                             | undefined |
| writable     | 可否被 assign                       | false     |
| enumerable   | 是否出現在 for ... in 或 Object.keys() | false     |
| configurable | 可否刪除                             | false     |

用於 Vue2
