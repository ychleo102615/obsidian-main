## Utility keyframes
教師常用的全域設定動畫。

```css
@keyframes fadeFromOpaque {
	from {
		opacity: 1;
	}
}
@keyframes fadeToOpaque {
	to {
		opacity: 1;
	}
}
@keyframes fadeFromTransparent {
	from {
		opacity: 0;
	}
}
@keyframes fadeToTransparent {
	to {
		opacity: 0;
	}
}
```
