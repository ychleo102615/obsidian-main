# spec-driven-development


# nimble #card  
[[*nem- "assign, allot; take"]] → niman（拿取）→ 反應快、手腳靈活  
靈活的、敏捷的  
能迅速「拿到」機會的人，手腳當然靈活  
`A nimble team can pivot their spec before the next sprint without ceremony.`


# consensus #card  
[[com-, intensive prefix or "with, together,"]] + [[*sent- "to go"]] → consentire（共同感受）→ 方向一致  
共識、一致意見  
所有人「感覺方向一致」就是 consensus  
`Without consensus on the spec's acceptance criteria, every stakeholder interprets "done" differently.`
